## 观察

对 Simulation 1（30×30 格点，20 个协变量 X ~ N(0,1)，
beta = (1,2,3,−4,−5,0_15)，2000 步 Gibbs，邻居平均模型），我估计 eta 标准误的
Cramér–Rao 下界约为 **0.11**；而 Table C1（eta = 0.4）报告 avgSEE = avgSEM =
0.044（ADVI）与 0.021（NUTS）——比该下界还小 **2.5–5 倍**。（随信附上的图
`fig_eta_se_floor.png` 显示:随协变量尺度下降,eta 的 SE 稳定在约 0.11,明显高于
论文报告的数值。)

我用四种彼此独立的方式做了核对，全部采用您的邻居平均约定与先验（Table C7：
b1=5, b2=50, c1=8, v0=1e-6）：

- GLM 条件伪似然（n=200）：eta 经验标准差约 0.32；
- 您的 spike-slab 模型经 ADVI（meanfield 与 fullrank 均试，n=200）：eta
  avgSEE 约 0.18–0.25；
- 同一模型经 NUTS（adapt_delta=0.99，无发散）：eta 经验标准差约 0.24，平均后验
  标准差约 0.26（8 个数据集）；
- 对正规 autologistic 联合分布做全似然蒙特卡洛 MLE（Geyer–Thompson）：其
  SE(eta) 与伪似然 SE 相等（比值约 1.0），在您的三个 eta 设置下皆然——也就是说
  伪似然已经达到了 Cramér–Rao 下界，不存在更高效的估计量。

关键在于：这个下界并不依赖协变量尺度——即便把协变量标准差一路降到接近 0，
eta 的 SE 也只降到约 0.11，从不接近 0.04。

## 一个相关的内部核对

Table C1 中 Proposed(ADVI)的 eta 行，avgBias = 0.087 约为 avgSEE = avgSEM =
0.044 的两倍，但 avgCR 却是 95%。若一个近似无偏的估计量标准差为 0.044，要达到
95% 覆盖，其偏差需远小于 0.044；而偏差达到约 2 倍标准误时，覆盖率会掉到约 50%。
反过来，要真正实现 95% 覆盖，真实标准差需在 0.13 左右——这与上面的 ~0.11 下界
吻合，而非与报告的 0.044 吻合。因此在我看来，avgSEM 列与 avgCR 列彼此不一致。

## 同样的模式也出现在 Simulation 2–3 与 HER2 分析中

这并非 Simulation 1 独有。在 Simulation 2 与 3 中，报告的 eta 经验标准误同样比
我得到的小 5–27 倍。真实数据分析中也是如此：论文报告 eta = 2.936，95% 区间
[2.926, 2.946]（宽度 0.02）；而我在同一份公开 HER2ST 数据上复现得到 eta = 2.665，
区间 [1.589, 3.999]（宽度约 2.4）——大约宽 120 倍。这正是我希望先弄清 SE / 可信
区间是如何计算的原因；点估计本身是接近的。

   0.24 的经验标准差算 SD/√200 约为 0.017，与报告的 0.021 接近——这也是我怀疑是
